# Artale 寵物補血設定 % 數計算器

這是一個專為 **Artale** 玩家設計的工具，用於精確計算寵物自動補血的設定比例。透過科學的公式計算，確保在練功或打王時，即使在延遲（Lag）或連續受傷的情況下也能保持安全血線。

## 📖 工具用途

在 Artale 中，寵物補血有一定的判定頻率。本工具透過「預留能撐幾下」的觀念，計算出最穩健的補血觸發百分比。

### 核心計算公式

<img width="282" height="69" alt="image" src="https://github.com/user-attachments/assets/b05a960e-94ff-40f4-8956-61dabd8cd257" />

* n：預留能撐幾下的額度。
* Hit：敵人最痛一下造成的傷害。
* HP：總血量（若勾選神聖之火，則為原始 ）。
* Heal：補品單次回復量。

---

## 🚀 功能特點

* **神聖之火切換**：自動處理龍騎士「神聖之火」技能後的 160% 血量加成。
* **風險判定機制**：
* **結果 < 100%**：正常顯示建議 % 數。
* **結果 > 100%**：觸發**紅色警告系統**，提示該補品或血量不足以支撐預期額度。


* **響應式設計**：支援手機與電腦瀏覽器開啟。

---

## 🛠️ 使用說明

1. 使用瀏覽器點選網址進行瀏覽。
2. 依序輸入您的原始血量、敵人傷害、補品量及預留次數。
3. 點擊 **「立即計算」** 即可獲得推薦設定值。

---

## 💡 實戰範例 (以龍騎士為例)

* **角色原始 HP**：9,000
* **敵人傷害**：1,400
* **補品**：熱狗堡 (回復 500)
* **預留額度**：3 下
* **狀態**：勾選神聖之火
* **計算結果**：**81.67%**

---

## ⚠️ 警告訊息說明

當計算結果顯示超過 100% 時，代表目前的補血策略不安全，工具會提供以下建議：

1. **可能需要玩家手動協助補血**（自動補血速度已跟不上）。
2. **調低預留次數**（現實面無法撐到這麼多下）。
3. **更換更高回復量的補品**（例如將熱狗堡更換為拉麵等更好的補品）。
